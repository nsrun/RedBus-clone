

# RedBus-clone

A web application that mimics the popular bus booking platform RedBus, built using Django for the backend and HTML/CSS for the frontend.

---

## Features

- User registration and login
- Search for buses between cities
- View available buses and seat selection
- Booking and ticket confirmation
- Admin panel for managing buses and bookings

---

## Technologies Used

- **Backend:** Python, Django
- **Frontend:** HTML, CSS
- **Database:** SQLite (default)
- **Other:** Bootstrap (optional for styling), JavaScript (for interactivity)

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/RedBus-clone.git
cd RedBus-clone
```

2. Create a virtual environment and activate it:

```bash
python -m venv env
source env/bin/activate  # On Windows: env\Scripts\activate
```

3. Install the dependencies:

```bash
pip install -r requirements.txt
```

4. Apply migrations and run the server:

```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```

5. Open your browser and go to [http://127.0.0.1:8000](http://127.0.0.1:8000)

---

## Usage

- Register as a user
- Search for buses by entering source, destination, and date
- Select a bus and choose seats
- Proceed to booking and payment (mock implementation)
- View your bookings in your profile
